Algumas Ferramentas "Free Software"

Baseadas no Código Livre

Os sistemas chamados "open sources" estão, a cada dia, crescendo na divulgação e no aperfeiçomento. Temos, ao nosso dispor, softwares isentos do pagamento de licenças, do mais alto nível técnico, que oferecem importantes recursos para a educação, comércio e indústria, sem prejuízo da segurança e eficiêcia. Existe vida inteligente fora do monopólio das empresas de software proprietário. A base do software livre é o  sistema operacional GNU/Linux. Existem várias distribuições de Linux, Vamos destacar  três opções: Ubuntu, kurumin (desktop) e Debian (server).  As opções de aplicativos, que rodarem debaixo  do Linux, são muitas, mas vamos citar duas: o pacote BrOffice e o Gimp. O BrOffice é composto de 6 aplicações, são elas: editor de texto e html (swriter), editor de apresentações, equivalente ao "PowerPoint" (impress), editor de planilhas (calc), editor gráfico (draw), editor de fórmulas matemáticas (math) e assistente de base de dados (base). 

Os sistemas open source contemplam os projetos sociais e educacionais.
